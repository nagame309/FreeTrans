# 截圖指南

為了讓 README 更完整,請添加以下截圖:

## 📸 需要的截圖

### 1. 浮動翻譯視窗
**檔名**: `screenshot-translation.png`
**內容**: 
- 在網頁上選取一段英文文字
- 顯示翻譯浮動視窗
- 展示毛玻璃效果和語言標籤

**建議尺寸**: 800x600 或更大

### 2. 設定面板
**檔名**: `screenshot-settings.png`
**內容**:
- 展示設定面板的完整介面
- 包含語言選擇、箭頭、開關等元素
- 顯示標題和 icon

**建議尺寸**: 400x500 左右

### 3. 使用示例 (可選)
**檔名**: `screenshot-demo.gif`
**內容**:
- GIF 動畫展示完整使用流程
- 選取文字 → 顯示翻譯 → 複製文字

**建議工具**: ScreenToGif, LICEcap
