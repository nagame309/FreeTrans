# Privacy Policy - 浮譯 FreeTrans

**Last Updated: 2026-01-29**

FreeTrans ("we", "us", or "our") is committed to protecting your privacy. This Privacy Policy explains how our Chrome extension handles information.

---

## 1. Information Collection and Use (資料收集與使用)

**FreeTrans does not collect, store, or transmit any personal identification information.**
(浮譯不會收集、儲存或傳送任何個人識別資訊。)

*   **User Data**: We do not collect your name, email address, browsing history, or any personal data.
*   **Translation Text**: The text you select for translation is sent directly to the translation service provider you have chosen (Google, Bing, OpenAI, or DeepL). We do not store this text or use it for any other purpose.
*   **API Keys**: If you provide your own API keys (e.g., OpenAI or DeepL), these keys are stored **locally** on your device using Chrome's secure storage sync/local. They are never transmitted to our servers.

## 2. Third-Party Services (第三方服務)

FreeTrans facilitates translation by connecting to the following third-party services. Please refer to their respective privacy policies for information on how they handle data:

*   **Google Translate**: [Google Privacy Policy](https://policies.google.com/privacy)
*   **Microsoft Bing**: [Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement)
*   **OpenAI**: [OpenAI Privacy Policy](https://openai.com/policies/privacy-policy)
*   **DeepL**: [DeepL Privacy Policy](https://www.deepl.com/privacy)

## 3. Data Security (資料安全)

Since all configurations and API keys are stored locally within your Chrome browser profile, your data is as secure as your local browser environment. We recommend using Chrome's built-in security features to protect your profile.

## 4. Changes to This Policy (政策變更)

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on our GitHub repository.

## 5. Contact Us (聯絡我們)

If you have any questions about this Privacy Policy, please contact us via our GitHub Issues page:
[https://github.com/nagame309/FreeTrans/issues](https://github.com/nagame309/FreeTrans/issues)

# 隱私權政策

**最後更新日期：2026-01-29**

浮譯 (FreeTrans) 承諾保護您的隱私。本政策說明了我們的 Chrome 擴充功能如何處理資訊。

## 1. 資料收集與使用

**浮譯不會收集、儲存或傳送任何個人識別資訊。**

*   **使用者資料**：我們不會收集您的姓名、電子郵件、瀏覽紀錄或任何個人資料。
*   **翻譯文字**：您選取用於翻譯的文字會直接發送至您選擇的翻譯服務提供者（Google、Bing、OpenAI 或 DeepL）。我們不會儲存這些文字，也不會將其用於任何其他目的。
*   **API 金鑰**：如果您提供自己的 API 金鑰（例如 OpenAI 或 DeepL），這些金鑰會透過 Chrome 的安全儲存機制（sync/local）儲存在您的**本機裝置**中。我們永遠不會將這些金鑰傳送到我們的伺服器。

## 2. 第三方服務

浮譯透過連接以下第三方服務來提供翻譯功能。關於他們如何處理資料，請參考其各自的隱私權政策：

*   **Google 翻譯**：[Google 隱私權政策](https://policies.google.com/privacy)
*   **Microsoft Bing**：[微軟隱私權聲明](https://privacy.microsoft.com/privacystatement)
*   **OpenAI**：[OpenAI 隱私權政策](https://openai.com/policies/privacy-policy)
*   **DeepL**：[DeepL 隱私權政策](https://www.deepl.com/privacy)

## 3. 資料安全

由於所有的設定與 API 金鑰都儲存在您的 Chrome 瀏覽器設定檔中，您的資料安全性取決於您的本機瀏覽器環境。

## 4. 聯絡我們

如果您對本隱私權政策有任何疑問，請透過我們的 GitHub 問題回報頁面聯絡我們。
